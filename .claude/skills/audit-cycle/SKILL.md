# Цикл аудита admik

КАМПАНИЯ УЖЕ СОШЛАСЬ (волна 17, цикл 11 = 0 настоящих багов).
БЕЗ повода/задачи владельца аудит НЕ запускать («не жги токены»).
Возобновлять только по явному слову владельца. Детали — `docs/10-состояние-и-продолжение.md`.

## Метод одного цикла
1. **Фан-аут:** 8 независимых финдеров по областям (деньги/оплата, доставка СДЭК, RBAC,
   токены/секреты, гонки/TOCTOU, медиа/S3, консистентность кеша, SQL/машины статусов).
2. **Дедуп** находок.
3. **2 независимых скептика** на КАЖДУЮ находку.
4. **САМОПРОВЕРКА по коду** каждой CONFIRMED-находки. Урок волны 17: ~⅓ «подтверждённых
   скептиками» оказались by-design/ложными. Фикс ложной находки = регрессия. Не доверять скептикам слепо.
5. **Фикс настоящего бага через ДВЕ проверки:** TDD-тест (скилл `gate`) + живой прогон на стенде
   (скилл `deploy-stand`, `verify-admin.mjs e2e`). Приоритет: деньги/безопасность — первыми.

## Критерий сходимости
Цикл с 0 настоящих багов = СХОДИМОСТЬ, кампанию закрыть. Тренд REAL по циклам логировать в `docs/00`.

## Итог
Сводка: находок → после дедупа → CONFIRMED скептиками → REAL после самопроверки → исправлено.
Каждый фикс — «код-гейт OK + живой прогон OK». Обнови `docs/00` и `docs/10`.
