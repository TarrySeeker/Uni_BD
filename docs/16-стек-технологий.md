# 16 — Стек технологий и языков проекта

> Полный перечень технологий, языков и инструментов, используемых в Admik
> (headless-бэкенд + админка) и в витрине-потребителе THE CASE. Версии — из
> `package.json` / `docker-compose.yml` на момент написания.

---

## Языки программирования

| Язык | Где используется |
|---|---|
| **TypeScript** (strict) | Основной язык приложения: бизнес-логика `lib/*`, роуты `app/*`, типы, серверные действия |
| **JavaScript / ESM** (`.mjs`) | Отдельные скрипты: `db/seed/owner.mjs`, `scripts/sharp-selfcheck.mjs`, конфиги (`next.config.mjs`, `eslint.config.mjs`, `postcss.config.mjs`) |
| **SQL** (PostgreSQL-диалект) | Идемпотентные миграции `db/migrations/*.sql`, параметризованные запросы (`postgres.js`) |
| **Bash** | DevOps-скрипты: `init-shop.sh`, `deploy.sh`, `backup.sh`, `restore.sh`, `update.sh`, `smoke.sh`, `check-migrations.sh` |
| **TSX / JSX** | React-компоненты админки и витрины |
| **CSS** (через Tailwind) | Стилизация; `globals.css` |
| **Dockerfile** | Многоступенчатая сборка рантайм-образа |
| **YAML** | CI (GitHub Actions `.github/workflows/ci.yml`), `docker-compose.yml`, Caddyfile-блоки |
| **JSON / JSONC** | Конфигурация (`tsconfig.json`, `package.json`, манифесты) |
